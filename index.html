Documentation

  1. Link hoverDisplay.js and hover_animate.css in your HTML file.

  2. Add class 'hoverMe' to desired trigger elements

  3. Add div class 'hoverInfo' as a sibling to 'hoverMe'. You can include as much or as little information in 'hoverInfo'.

  4. Wrap 'hoverMe' and 'hoverInfo' in a 'hoverContainer' or 'hoverContainer-abs' div. Use the latter if you wish to position your elements via CSS positioning.

  5. You choose intro/outro options utilizing existing built-in animations and speed settings.

  6. In your Javascript file, include the following code to call the function. 

    
$(function() {
	var settings = {
		intro: "fadeIn",
		outro: "fadeOut",
		moveY: 0,
		moveX: 0,
		speed: "slow"
	}

	$( ".hoverMe" ).hoverDisplay(settings);
});
